Saving & loading your plan

No accounts, no cloud — the whole plan lives in one small file you keep.

The live link (the easy way)

Hit Start syncing (top right), name the crew, and you'll land on a special link — that link is the plan. Drop it in the group chat: everyone who opens it sees the same adventures and can edit them, live, from any device. No accounts, no passwords. Two things to know: anyone with the link can edit or delete anything (deletions are kept recoverable behind the scenes), so don't post it publicly — and don't put home addresses or payment details in the plan.

  1. Backup (top right) downloads tpc-adventures-plan.json — everything on this page in one file.
  2. Keep it where the crew can find it — the group chat, a shared Drive, anywhere.
  3. Restore brings it all back — on any device, in any browser.
💻

Laptop

  • Backup → the file lands in your Downloads folder.
  • Restore → a file picker opens; grab tpc-adventures-plan.json.
  • Tip: drag the file straight into the group chat to share the latest version.
📱

iPhone & iPad

  • Safari asks “Do you want to download?”, then shows a blue ↓ by the address bar.
  • The file lives in the Files app → Downloads.
  • Restore → tap Choose File → Browse → Downloads and pick it.
🤖

Android

  • Chrome shows a download bar at the bottom when you hit Backup.
  • Find the file in Downloads (or the Files app).
  • Restore → the picker opens; check Recent or Downloads.

The file is tiny — send it in the group chat and anyone can restore it. Heads-up: restoring a file replaces what's on screen, so hit Backup first if you've made changes you want to keep. On a live crew link, backups double as snapshots you can keep before big edits.
