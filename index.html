Saving & loading your plan

No accounts, no cloud — the whole plan lives in one small file you keep.

  1. Save plan (top right) downloads tpc-adventures-plan.json — everything on this page in one file.
  2. Keep it where the crew can find it — the group chat, a shared Drive, anywhere.
  3. Load plan brings it all back — on any device, in any browser.
💻

Laptop

  • Save → the file lands in your Downloads folder.
  • Load → a file picker opens; grab tpc-adventures-plan.json.
  • Tip: drag the file straight into the group chat to share the latest version.
📱

iPhone & iPad

  • Safari asks “Do you want to download?”, then shows a blue ↓ by the address bar.
  • The file lives in the Files app → Downloads.
  • Load → tap Choose File → Browse → Downloads and pick it.
🤖

Android

  • Chrome shows a download bar at the bottom when you hit Save.
  • Find the file in Downloads (or the Files app).
  • Load → the picker opens; check Recent or Downloads.

The file is tiny — send it in the group chat and anyone can load it. Heads-up: loading a file replaces what's on screen, so hit Save plan first if you've made changes you want to keep.
